# Unit test execution
* To run all the unit tests,
    ```shell
    fx set core.qemu-x64 --with-host //src/testing/end_to_end/honeydew/tests/unit_tests:tests
    fx test //src/testing/end_to_end/honeydew/tests/unit_tests --host --output
    ```
